## 📖 Overview
This project implements a **Parking Slot Availability Indicator** using **combinational logic circuits**. The system monitors four parking slots using SPDT switches as sensors and provides visual feedback via LED indicators. It operates in two modes:
- **Normal Mode**: Displays status of all slots simultaneously.
- **Priority Mode**: Highlights only the first available slot based on a fixed priority order.

---

## 🛠️ Features
- Real-time slot status (Empty/Occupied)
- **AnyEmpty** indicator (lights up when at least one slot is free)
- **Full** indicator (lights up when all slots are occupied)
- Priority-based slot suggestion
- Designed and simulated using **Multisim**
- Boolean logic simplification using DeMorgan’s Law
